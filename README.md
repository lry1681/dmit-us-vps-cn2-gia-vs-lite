## DMIT的四个系列，到底有啥区别?

先说清楚DMIT美国机房的产品线结构。目前一共四个系列，核心差异在**线路**和**价格**：

### 1. **Lite系列** - 普通线路，月付$6.9起

这是DMIT最便宜的系列。用的是**普通国际线路**，不是CN2，更不是GIA。但胜在带宽给得足，流量也不小气。自带1个IPv4和1个IPv6/64。

**适合谁？**  
预算紧张、对延迟要求不高的场景，比如挂个机器人、跑个测试环境，或者纯粹当备用机。

### 2. **Premium系列** - CN2 GIA线路，季付$28.88起

这是主力产品。基于**AMD EPYC处理器**，走的是CN2 GIA线路。三网回程都是AS4809的GIA，速度和稳定性有保障。默认也是1个IPv4 + 1个IPv6。

**适合谁？**  
需要稳定访问中国大陆的网站、应用部署，或者对延迟敏感的业务。性价比在GIA系列里算高的。

### 3. **Premium Secure** - 高防GIA线路

这个系列的特点是**去程加了Cloudflare Magic Transit的DDoS防护**，回程依然是三网GIA（AS4809）。目前只有一个套餐可选。

容易被攻击的业务，比如游戏服务器、对外API接口。花钱买个安心。

### 4. **Premium Unmetered** - 不限流量的GIA线路

顾名思义，**流量不限制**。同样是AMD EPYC + CN2 GIA组合，配置拉满，但价格也是真的贵。

**适合谁？**  
流量需求大到没边的场景，比如视频站、大文件分发。普通用户基本用不上。
